### How do I get set up? ###

* sudo mysql --password
* create database unidb;
* cd to the dir conatining db_structure.sql
* run  mysql -h localhost -u root -p unidb < db_structure.sql


### How do i perform a simple api test? ###


* cd to the root directory
* run mvn spring-boot:run 


* visit
* http://localhost:8080
